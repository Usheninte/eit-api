# API

| Path          | Action                    |
| ------------- | ------------------------- |
| `/`           | [READ] Get all EITs       |
| `/add`        | [CREATE] Add an EIT       |
| `/update/:id` | [UPDATE] Edit EIT by ID   |
| `/delete/:id` | [DELETE] Delete EIT by ID |
